# Niyantra - Intelligent Asset Management Software

## Overview
**Niyantra** is a comprehensive, web-based IT asset management solution designed to help modern enterprises track, manage, and optimize their hardware lifecycle. From acquisition to disposal, Niyantra ensures complete visibility and accountability for all your organization's assets.

## Key Features

### 1. Complete Lifecycle Tracking
*   **Acquisition to Disposal**: Monitor every stage of an asset's life.
*   **Real-time Status**: Instantly know if an asset is In Stock, Assigned, In Repair, or Scrapped.
*   **Warranty & Depreciation**: Track warranty expiry dates and calculate asset depreciation automatically.

### 2. Smart Asset Transfers
*   **Seamless Movement**: Transfer assets between different companies, branches, or locations with ease.
*   **Audit Trail**: Automatically log every transfer to maintain a complete history of custody.
*   **Employee Assignment**: Assign assets to specific employees and track return dates.

### 3. Multi-Company & Multi-Location Support
*   **Unified Dashboard**: Manage multiple legal entities (companies) under a single system.
*   **Branch Management**: Organize assets by specific branches and physical locations.
*   **Scalable**: Suitable for small businesses and large enterprises with distributed teams.

### 4. Advanced Analytics & Dashboard
*   **Visual Insights**: Interactive charts and graphs showing asset distribution, status breakdowns, and acquisition trends.
*   **At-a-Glance Metrics**: Key performance indicators (KPIs) for total assets, active assets, and maintenance alerts.

### 5. Comprehensive Reporting
*   **Custom Reports**: Generate detailed reports based on specific criteria (e.g., "All laptops in New York branch").
*   **Excel Export**: Export full data sets to Excel (.xlsx) for offline analysis and auditing.
*   **Audit Compliance**: Ensure you have the data needed for financial and security audits.

### 6. User Management & Security
*   **Role-Based Access**: Define roles (Admin, Staff, Viewer) to control what users can see and do.
*   **Secure Authentication**: Powered by robust authentication protocols to keep your data safe.

## Technology Stack
Niyantra is built using modern, high-performance technologies:
*   **Frontend**: React.js (Vite) for a fast, responsive user interface.
*   **Styling**: Tailwind CSS for a modern, clean, and mobile-friendly design.
*   **Backend**: Firebase (Firestore) for real-time database capabilities.
*   **Visualization**: Recharts for dynamic data visualization.

## Getting Started
To start using Niyantra:
1.  **Login**: Access the secure dashboard using your credentials.
2.  **Add Assets**: Use the "Add Asset" form to register new hardware.
3.  **Manage**: Use the "Assets" list to edit, transfer, or update the status of items.
4.  **Analyze**: Check the Dashboard for insights and use Reports for detailed data.
